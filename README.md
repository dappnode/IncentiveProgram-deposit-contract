# Dappnode Incentive program contracts

## Requirements

- node version: 14.x
- npm version: 7.x

## Install

```
npm i
```

## Run tests

```
npm run test
```

## Deploy on hardhat

```
npm run deploy:IncentiveProgram:hardhat
```

## Deploy on gnosis

```
npm run deploy:IncentiveProgram:gnosis
```
